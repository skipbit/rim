# Gemini Project Context: rim

## Project Goal
A minimal, console-based text editor similar to NeoVim, built in Rust.

## Commands
- **Build:** `cargo build`
- **Run:** `cargo run -- <filename>` (or `target/debug/rim <filename>` after building)
- **Test:** `cargo test`
- **Lint:** `cargo clippy`
- **Format Check:** `cargo fmt -- --check`
